# File Processing System

## Apresentação do Projeto

Este projeto, **File Processing System**, tem como objetivo receber arquivos, processar seus dados e gerar um relatório. Esse projeto futuramente pode ser moldado para diversos ramos, entretanto a princípio ele será utilizado para gerar relatório de um mercado. 

## Arquitetura

A arquitetura adotada neste projeto é a **Arquitetura Em Camadas (Layered Architeture)**. Irei utilizar pois é muito simples de se entender e muito se vê em empresas este método de arquitetura


### Estrutura do Projeto:

```
FPS
FileProcessor/
├── Presentation/        # Entrada do sistema (console, worker, etc)
│
├── Application/         # Regras de negócio / serviços
│
├── Domain/              # Entidades (modelo do sistema)
│
├── Infrastructure/      # Acesso a arquivos, banco, etc
│
└── FileProcessor.csproj
```

## Metodologia Utilizada

Utilizei metodologia ágil com Scrum, organizando o desenvolvimento em sprints curtas e priorizando entregas incrementais.
